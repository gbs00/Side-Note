### 详细描述

**📝 Side Note — 专为浏览而生的轻量笔记**

在浏览网页时突然有了灵感？打开 Side Note，几秒内开始记录。

• **即开即用** — 点击图标，侧边栏秒开，无需跳转新页面
• **Markdown 实时渲染** — 标题、加粗、列表、代码...所见即所得
• **隐私优先** — 数据仅存本地，关闭标签页自动清除，零追踪
• **深色模式** — 自动跟随系统，护眼更舒适
• **一键复制** — 导出完整笔记，含网页链接和时间戳

**🎯 适用场景**

• 在线学习时记录要点
• 浏览文章时摘抄精彩内容
• 产品调研时快速记录灵感
• 会议期间随手记笔记

**🔒 隐私承诺**

Side Note 不收集任何个人信息。笔记内容仅存储于浏览器本地，关闭标签页后自动删除，我们无法访问您的任何数据。

## English Version

### Short Description (under 132 characters)
Lightweight note-taking in browser sidebar — Markdown live preview, auto-clears on tab close, zero tracking.

### Detailed Description

**📝 Side Note — Effortless Notes While You Browse**

Got an idea while browsing? Open Side Note and start writing in seconds.

**✨ Key Features**

• **Instant Access** — Click the icon, sidebar opens instantly, no new tabs
• **Live Markdown** — Headings, bold, lists, code...rendered as you type
• **Privacy First** — Data stays local, auto-clears on tab close, zero tracking
• **Dark Mode** — Follows system preference automatically
• **One-Click Copy** — Export full note with URL and timestamp

**🎯 Perfect For**

• Taking notes while learning online
• Saving quotes from articles
• Jotting down ideas during research
• Quick notes during meetings

**🔒 Privacy Promise**

Side Note collects zero personal data. Notes are stored only in your browser's local storage and automatically deleted when you close the tab. We cannot access any of your data.
