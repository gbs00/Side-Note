# Side Note 0.1.1 版本测试计划

## 版本信息
- **版本号**: 1.0.0
- **测试日期**: 2026-02-04（首轮） / 2026-02-10（复测）
- **测试范围**: 数据持久逻辑优化 + UI/UX 改进

## 更新内容概览

### 【1】数据持久逻辑优化
- **同一 tab**: 刷新/URL 跳转 不清空，内容保持
- **克隆 tab**: 新 tab 新建空白 note（完全独立）
- **关闭 tab**: 立即清空该 tab 的 note
- **关闭窗口**: 窗口内所有 tab 触发清空
- **退出浏览器**: chrome.storage.session 天然会话级清空

### 【2】UI/UX 优化
- 新设计稿样式（ui20260203.pen）
- 玻璃拟态背景 + 现代卡片式设计
- 空状态显示（✦/标题/描述）
- 引用块竖线样式
- 主题切换图标优化
- Toast 提示样式优化
